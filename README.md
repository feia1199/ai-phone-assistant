# AI Phone Assistant 🤖

AI 手机自动化助手 - 一句话让 AI 帮你操作手机

## ✨ 功能特性

### 🎯 核心功能
- **无障碍读屏** - 读取任意 App 的界面元素
- **AI 指令解析** - 自然语言 → 自动生成操作步骤
- **自动操作** - 点击、输入、滑动、返回、回桌面
- **打开 App** - 内置 18 个常用 App 快捷打开

### 🎙️ 语音系统
- **语音输入** - 说话下达指令，自动执行
- **语音播报** - 执行结果语音反馈
- **多声音风格** - 支持多种语音风格切换
  - 🎀 **豆包** - 年轻女性，活泼亲切（默认）
  - 🌸 **小燕** - 温柔女声，优雅舒缓
  - 🎩 **小风** - 成熟男声，稳重可靠
  - 🧸 **可爱** - 可爱童声，俏皮活泼

### 🫧 悬浮窗
- 任意界面拖动、展开控制面板
- 快速语音指令
- 一键回桌面/返回

## 📱 使用步骤

1. **安装 APK** → 允许未知来源安装
2. **开启权限** → 无障碍权限 + 悬浮窗权限
3. **选择语音风格** → 默认豆包，可切换其他声音
4. **配置 API Key** → 填入 OpenAI/DeepSeek/智谱的 API Key
5. **开始使用** → 语音或文字输入指令 → AI 自动执行

## 🎙️ 支持指令

- `打开微信` / `打开支付宝` / `打开淘宝`
- `打开设置` / `回到桌面` / `返回`
- `向上滚动` / `向下滚动`
- 更多自然语言指令...

## 🎨 语音风格

### 豆包风格（默认）
- 年轻女性声音
- 音调略高，活泼亲切
- 适合日常助手场景
- 示例："你好呀！我是你的 AI 助手，有什么可以帮你的吗？"

### 小燕风格
- 温柔女声
- 语速舒缓，优雅
- 适合正式场合

### 小风风格
- 成熟男声
- 稳重可靠
- 适合商务场景

## 🔧 技术栈

- Kotlin + Android Jetpack
- AccessibilityService（无障碍服务）
- OpenAI/Claude API（AI 大脑）
- Text-to-Speech（语音合成）
- SpeechRecognizer（语音识别）
