# AI 手机助手 - 使用说明

## 前置条件

1. **Android Studio**（最新版）
2. **安卓手机**（Android 7.0+）
3. **AI API Key**（OpenAI / Claude / 国产大模型均可）

## 编译安装

### 1. 用 Android Studio 打开项目
```
File → Open → 选择 ai-phone-assistant 文件夹
```

### 2. 等待 Gradle 同步完成
首次打开会自动下载依赖，需要几分钟。

### 3. 连接手机
- 手机开启「开发者选项」→「USB 调试」
- 用 USB 线连接电脑
- 手机上授权调试

### 4. 编译运行
点击 Android Studio 的绿色 ▶ 按钮即可安装到手机。

---

## 使用流程

### 第一步：开启无障碍权限
1. 打开 App → 点击「开启无障碍权限」
2. 在系统设置中找到「AI 助手」→ 开启
3. 返回 App，状态变为 ✅

### 第二步：配置 API Key
- 填入你的 OpenAI API Key
- 如果用其他模型，修改 API 地址：
  - Claude: `https://api.anthropic.com/v1/messages`（需改代码适配）
  - 智谱: `https://open.bigmodel.cn/api/paas/v4/chat/completions`
  - DeepSeek: `https://api.deepseek.com/chat/completions`

### 第三步：下达指令
输入自然语言指令，例如：
- `打开微信`
- `打开设置`
- `回到桌面`

---

## 支持的操作

| 指令 | 说明 |
|------|------|
| `打开微信` | 打开微信 App |
| `打开淘宝` | 打开淘宝 App |
| `回到桌面` | 按 Home 键 |
| `返回` | 按返回键 |

更多操作需要 AI 根据屏幕内容动态生成。

## 架构说明

```
用户输入指令
    ↓
获取屏幕无障碍树信息
    ↓
发送给 AI API 分析
    ↓
AI 返回操作指令列表
    ↓
逐步执行操作（点击/输入/滚动）
    ↓
返回结果
```
